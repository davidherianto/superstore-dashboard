# Superstore Performance Analysis

This project analyzes the sales performance of a retail company using the Superstore dataset. The analysis focuses on identifying four KPIs (Revenue, Profit, Orders, and Profit Margin) across different regions, product categories, and customer segments.

## A. Software Tools
- SQL
- PowerBI

## B. Business Question
- How has business performance changed over time?
- Which regions, product categories, and customer segments contribute most to sales and profit?

## C. Dataset Overview
- The dataset can be downloaded from the following link:  
[Superstore Dataset (Kaggle)](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
- The dataset contains 9984 row and 21 columns.

## D. Data Preprocessing
- Missing Value  
No missing values were found in this dataset.

- Duplicate Value  
During the data quality assessment, duplicate records were identified at Row_ID 3406 and Row_ID 3407. All values across the columns in both rows were identical, indicating a duplicated entry. To prevent potential bias and ensure the accuracy of the analysis, one of the duplicate records was removed.

- Outlier Value  
No outlier values were found in this dataset.
